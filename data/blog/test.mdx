# 欢迎来到我的博客

这是我的第一篇博客，主要介绍了如何使用 Next.js 和 Tailwind CSS 搭建个人博客。

## 1. 安装依赖

首先，我们需要安装项目依赖：

npm install

## 2. 启动项目

安装完成后，使用以下命令启动项目：

next dev

## 3. 撰写博客

在 `/data/blog` 目录中新建 `.mdx` 文件，开始撰写你的博客吧！
